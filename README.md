# Telegram бот: Яндекс.Музыка трек-инфо

Бот принимает ссылку(и) на трек в Яндекс.Музыке и возвращает красиво отформатированную информацию:
- название трека
- артист(ы)
- длительность (мм:сс)
- альбом (если есть)
- год
- ссылка на трек

## Установка

### 1) Создай виртуальное окружение (рекомендуется)

**Windows (PowerShell):**

```bash
py -m venv .venv
.\.venv\Scripts\Activate.ps1
```

### 2) Установи зависимости

```bash
pip install -r requirements.txt
```

## Настройка

1) Скопируй файл `.env.example` в `.env`
2) Впиши токен бота:

```env
BOT_TOKEN=123456:ABCDEF_your_token_here
```

Опционально можно указать `YM_TOKEN` (токен Яндекс.Музыки). Для части публичной информации он может не требоваться, но если данные не подтягиваются — добавь токен.

Если в твоей сети Telegram не открывается и нужны прокси — добавь в `.env` переменную `PROXY_URL` (HTTP или SOCKS5). Зависимость для прокси уже есть в `requirements.txt`.

## Запуск

```bash
python main.py
```

## Использование

- Команда: `/start`
- Ссылки:
  - `https://music.yandex.ru/track/123456789`
  - `https://music.yandex.ru/album/111/track/222`
- Можно отправлять несколько ссылок одним сообщением.

## Структура

- `main.py` — входная точка
- `config.py` — загрузка `.env`
- `handlers/` — обработчики `aiogram`
- `utils/` — парсинг ссылок и получение данных Я.Музыки
